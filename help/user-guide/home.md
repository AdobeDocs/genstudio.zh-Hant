# GenStudio使用手冊

Adobe GenStudio採用generative AI，革新了內容供應鏈，讓您建立、測量和管理體現品牌精髓的個人化體驗。

使用GenStudio，行銷人員和團隊可以輕鬆產生並調整符合品牌准則的高品質內容。 GenStudio可加快行銷生命週期，節省寶貴的時間和資源。
