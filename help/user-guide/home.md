# GenStudio使用手冊

GenStudio是全方位的行銷解決方案，由Adobe的創作AI技術提供支援。 它讓行銷人員能夠簡化內容供應鏈，從構思到交付和績效測量。

透過GenStudio，企業可以利用創作AI來加速內容建立、輕鬆尋找及重複使用內容，並推動成功的全通路行銷活動。
