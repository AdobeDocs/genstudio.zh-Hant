# 高效能行銷人員Adobe GenStudioBeta發行說明

這些附註著重指出，在截至9月19日的一週中，效能行銷人員修正和增強功能的重大Adobe GenStudio。

## 新功能和增強功能

* **與Adobe Experience Manager Assets整合**。 現在提供Adobe Experience Manager Assets的唯讀存取權。<!-- GS-2432 -->

* **更新範本工作流程的增強功能**。  現在，更新範本的使用者會選取他們想要使用範本的頻道。<!-- GS-4029 -->

* **已改善建立頁面載入效能**。 未使用的相依性已從頁面載入程式中移除。<!-- GS-3630 -->

* **多節電子郵件支援**。 編輯人員現在可以產生包含多個區段和影像的電子郵件。 他們也可以重新產生產生產生之電子郵件的特定片段（例如標題）。<!-- GS-2436 -->

* **建立期間**&#x200B;在案頭和行動檢視之間切換。 使用者現在可以在案頭和行動檢視之間切換，以預覽電子郵件體驗變體。<!-- GS-4314 -->

* 內容現在會產生裁切尺寸與原始資產尺寸相關的影像。<!-- GS-3150 -->

* 使用者現在可以選取產生的影像變體，並在建立工作流程期間使用「調整裁切」功能來裁切。<!-- GS-5538 2342 -->

* 核准體驗的詳細資訊檢視現在會顯示縮圖影像，以及該體驗中參照之所有資產的狀態。<!-- GS-3783 -->

## 已知問題

下列已知問題排程在GenStudio for Performance Marketers GA版本中解決。

* 間歇性延遲問題會影響某些[!DNL Create]畫布作業。<!-- GS-5203 -->

* 產生電子郵件會導致電子郵件不完整。 **因應措施**：重新整理頁面並重新產生。<!-- GS-5209 -->

* 可以上傳範本但無法看到範本。 **因應措施**：上傳已填入&#x200B;**[!UICONTROL 行銷活動]**&#x200B;欄位的資產。 然後，再次上傳範本。<!-- GS-4815 -->

* 使用者也必須登入Facebook，才能登入管道中繼廣告帳戶兩次。 **因應措施**：登入Channel Meta ads帳戶之前，請先登出Facebook。<!-- GS-4806 -->

### 其他增強功能和已修正的問題

* 現在，拖放功能在提示區域中可如預期運作。<!-- GS-3977 -->

* 修正使用Tab鍵導覽左側導覽列上的元素的問題。 之前，您必須按多次才能從一個元素導覽至下一個使用中元素。 <!-- GS-2639 -->

* 現在，當使用者在體驗載入時編輯體驗名稱時，GenStudio會儲存體驗名稱。<!-- GS-5242 -->

* 使用者現在可以成功編輯體驗標題。 之前，使用者嘗試編輯標題文字後，標題文字會預設為原始文字。<!-- GS-5246 -->
* 在多部分電子郵件建立期間，選取的影像現在會如預期般呈現在畫布上。<!-- GS-5263 -->

* [!DNL Content]體驗詳細資訊頁面中的所有字串現在都已本地化。<!-- GS-5016 -->

* 使用者現在可以刪除在[!DNL Products]中開啟其詳細資料檢視的產品。<!-- GS-5057 -->

* 已改善搜尋未產生相符結果時GenStudio顯示的訊息。<!-- GS-4544 -->

* 搜尋篩選值的`aria-label`屬性值現在已如預期轉譯。<!-- GS-5388 -->

* 使用者現在可以成功刪除[!DNL Create]畫布提示區域中的重複資產。 <!-- GS-5233 -->

* 帳戶篩選器現在可如預期般處理體驗、資產和屬性。<!-- GS-4812 -->

* 中繼廣告範本的字型問題已解決，以提高可讀性和可存取性。<!-- GS-5354 -->

* 草稿標題的變更現在會如預期般持續存在。 之前，標題在編輯後會回覆為預設名稱。<!-- GS-2951 -->

#### 範本修正

* 調整大小功能現在可以如預期在中繼廣告範本中使用多個影像。 之前，GenStudio不會為所有選取的範本調整影像大小。<!-- GS-4696 -->

* 刪除範本現在會如預期重新整理[!DNL Content]頁面。<!-- GS-5397 -->

* 使用者現在只能從下拉式功能表中選取[!DNL Personas]、[!DNL Brands]或[!DNL Products]的值。 先前，_範本上傳_&#x200B;對話方塊可讓使用者輸入任何[!DNL Persona]、[!DNL Brand]或[!DNL Product]名稱。<!-- GS-5072 5071-->

* **[!UICONTROL 上一步]**&#x200B;按鈕現在已在範本上傳過程中停用。<!-- GS-5358 -->

* [!DNL Create]選取範本詳細資料檢視中的所有字串現在都已本地化。<!-- GS-5025 -->

## Beta舊版

舊版Beta包含下列重點和修正。

### 反白顯示

* [!DNL Create]內容選擇器已重構，以改善資產載入狀況。<!-- GS-2586 -->

* GenStudio現在支援在[!DNL Insights]表格和相簿檢視中預覽媒體資產的選項。 視訊縮圖包含具有靜音選項的&#x200B;**播放**&#x200B;按鈕。<!-- GS-4398 -->

* instagram和Facebook管道指引已合併至中繼品牌指引。

* 已簡化[!DNL Create]個畫布導覽元素。 [!DNL Create]登陸頁面會顯示左側導覽面板，但使用者現在會使用「**[!UICONTROL 上一步]**」按鈕，從其他[!DNL Create]個工作區域導覽至此分享空間。

* 導覽元素已增強，可在執行整個產品任務時支援使用者焦點，包括下列產品區域：

   * [!DNL Content]中的資產、體驗、範本詳細資料
   * [!DNL Insights]中的體驗、資產、屬性詳細資料
   * [!DNL Brands]中的品牌詳細資料
   * 產品和角色中的產品和角色詳細資訊

* 使用者不再需要按一下&#x200B;**[!UICONTROL 重新整理]**&#x200B;按鈕，即可檢視[!DNL Content]中體驗的更新。

* _體驗詳細資料_&#x200B;頁面現在會將外部資產縮圖轉譯為HTML。

* 新增或刪除Assets和體驗後的UI延遲已有所改善。

* 範本預覽現在包含更多描述性預設文字。 請參閱[自訂範本](https://experienceleague.adobe.com/en/docs/genstudio/user-guide/content/templates/customize-template#template-preview)。

* **以百分比為基礎的驗證分數**：品牌驗證現在會以百分比顯示品牌驗證分數，而非通過/失敗值。 （固定8/16）

* **已更新品牌擷取介面**：品牌擷取現在會以百分比顯示擷取流程的完成。 （固定8/16）

* **在擷取期間遞增品牌載入**：品牌指引現在以遞增方式載入使用者介面。 （固定8/16）

* **多節電子郵件建立**：使用者現在可以建立包含個別標題、影像、內文和CTA元素的電子郵件。 （固定8/16）

* **中繼廣告調整大小**：編輯人員可以調整中繼廣告外觀比例。 （固定8/16）

* **有限的[!DNL Insights]登入帳戶**： [!DNL Insights]登入現在僅支援每個客戶一個帳戶。 （固定8/16）

### 其他增強功能和已修正的問題

* [!DNL Create]畫布現在可正確轉譯中繼廣告中的影像。 （固定9/13） <!-- GS-4864 -->

* 雖然中繼廣告畫布預覽和匯出的檢視之間可能存在差異，但匯出的體驗會如預期般運作。 （固定9/13） <!-- GS-4492 4401 -->

* 上傳的影像現在包含預期的智慧標籤。 （固定9/13） <!-- GS-4856 -->

* 中繼廣告匯出CSV檔案現在包含如預期的影像。 之前，ZIP檔案包含CSV匯出檔案和NULL檔案，而非影像。 （固定9/13） <!-- GS-5107 -->

* 使用者現在可以如預期在[範本詳細資訊]檢視&#x200B;**[!UICONTROL 上載者]**&#x200B;欄位中輸入文字。 之前，載入圖示會防止使用者輸入文字。 （固定9/13） <!-- GS-4887 -->

* 刪除品牌後，使用者不再重新導向至品牌的詳細資料檢視。 （固定9/13） <!-- GS-2663 -->

* 編輯人員傳送變體以供稽核及核準時，不再收到下列錯誤： `You have no access to view comments on this Object`。 （固定9/13） <!-- GS-5140 -->

* 更新檢閱和核准工作流程使用的電子郵件範本。 （固定9/13） <!-- GS-5239 -->

* 現在當範本選擇器載入期間發生網路錯誤時，GenStudio會顯示錯誤訊息。 （固定9/13） <!-- GS-4682 -->

* 已解決從資產、體驗或範本卡片導覽至所選物件時的問題。 （固定9/13） <!-- GS-4390 -->

* 從「建立畫布」開啟時，_新增Assets_&#x200B;快顯視窗現在已本地化。 （固定9/13） <!-- GS-4867 -->

* 再生的變體現在會觸發品牌驗證。 以前，如果編輯者重新產生現有草稿的變體，則不會觸發驗證。 （固定9/13） <!-- GS-3971 -->

* _新增Assets_&#x200B;快顯視窗現在已如預期般本地化。 （固定9/5） <!-- GS-3834 -->

* 中繼廣告體驗範本的縮放問題已解決。 （固定9/5） <!-- GS-4174 -->

* 多部分電子郵件的CSV匯出檔案中的文字欄位現在會依預期排序。 （固定9/5） <!-- GS-4013 -->

* 當使用者重複按下&#x200B;**退格鍵**&#x200B;以清除搜尋欄位文字時，[!DNL Content]搜尋欄位不再消失。 （固定9/5） <!-- GS-4543 -->

* 當共同作業人員將`@`提及新增至評論時，適用於效能的GenStudio行銷人員現在會依預期載入使用者。 先前未載入使用者，且顯示錯誤： `Unable to load users. Refresh the page`。 （固定8/29） <!-- GS-4113 -->

* 在提示區域的電子郵件建立期間，當編輯者按一下&#x200B;**選取內容**&#x200B;時，GenStudio不再顯示&#x200B;**發生錯誤**&#x200B;訊息。<!-- GS-4879 -->

* _體驗詳細資料_&#x200B;頁面摘要位置名稱現在會指定Facebook或Instagram摘要。 （固定8/16）

* 現在，當使用者從&#x200B;_資產總覽_&#x200B;檢視導覽至&#x200B;_資產詳細資訊_&#x200B;檢視時，裁切較大的影像和視訊會保持一致。 （固定8/16）

* 在使用者登入之前，屬性畫面搜尋結果計數不再顯示`0 of`。 （固定8/16） <!-- GS-3665 -->

* 按一下&#x200B;**[!UICONTROL [!DNL Insights]]** > **[!UICONTROL 資產]**&#x200B;計數欄位不再清除搜尋和篩選設定。 （固定8/16） <!-- GS-3476 -->

* 當使用者嘗試在[!DNL Insights]檢視中輸入認證時，GenStudio會顯示錯誤。 （固定8/29） <!-- GS-4689 -->
